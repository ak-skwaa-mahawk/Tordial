# 3. Recovery Stages Overview

Deep recovery proceeds through **six stages**:

1. **Containment** — stop all autonomous activity  
2. **Stabilization** — restore geometric stillness  
3. **Re‑Contraction** — rebuild Carroll Rings convergence  
4. **Re‑Resonance** — rebuild Mesh coherence  
5. **Re‑Phase** — restore π₃D alignment  
6. **Re‑Identity** — re‑establish sovereign identity  

The organism must pass through all six stages in order.

Skipping a stage risks permanent destabilization.

# 4. Stage 1 — Containment

## Objective
Stop all autonomous subsystems to prevent further geometric damage.

## Required Conditions
- override_lambda = 1  
- Vault sealed  
- Planner halted  
- Walker halted  
- Mesh locked  

## Operator Actions
1. Confirm HITL override is active  
2. Do **not** attempt to resume  
3. Do **not** issue Slow or Freeze  
4. Ensure no external inputs are being injected  
5. Ensure no operator tools are forcing stride/horizon  

## Success Criteria
- system enters full stillness  
- no drift changes for 3 cycles  
- no Mesh amplitude changes  

Containment is complete when the organism is **quiet**.

## Objective
Allow the manifold to stop deforming.
