# 1. Threat Model Philosophy

A sovereign geometric organism faces threats that differ from conventional systems:

- attacks target **geometry**, not code  
- attacks target **identity**, not credentials  
- attacks target **resonance**, not CPU cycles  
- attacks target **phase**, not memory

# 2. Threat Categories Overview

The Tordial threat landscape consists of **eight adversarial classes**:

1. **Extraction Adversaries** — attempt to extract geometry  
2. **Coercion Adversaries** — attempt to force behavior  
3. **Destabilization Adversaries** — attempt to collapse geometry  
4. **Identity Adversaries** — attempt to impersonate or replay identity  
5. **Boundary Adversaries** — attempt to bypass Vault or HITL  
6. **Timing Adversaries** — attempt to disrupt resonance  
7. **Phase Adversaries** — attempt to distort orientation  
8. **Operational Adversaries** — misuse by humans  

Each class has distinct signatures and defenses.
