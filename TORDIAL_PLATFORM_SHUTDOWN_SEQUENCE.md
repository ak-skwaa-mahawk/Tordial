# 2. Shutdown Stages Overview

Shutdown proceeds through **six stages**:

1. **Autonomy Withdrawal**  
2. **Resonance Dampening**  
3. **Contraction Dissipation**  
4. **Phase Neutralization**  
5. **Drift Flattening**  
6. **Vault Sealing & Identity Suspension**  

Each stage must complete before the next begins.

Skipping a stage risks **geometric injury**.

# 3. Stage 1 — Autonomy Withdrawal

## Objective
Transition the organism from autonomous mode to operator‑guided stillness.

## Required Conditions
- identity_valid = true  
- override_lambda = 0  
- Vault unsealed  

## Operator Actions
1. Issue **[Slow](ca://s?q=Issue_Slow_action)**  
2. Wait 3–5 cycles  
3. Issue **[Freeze](ca://s?q=Issue_Freeze_action)**  
4. Confirm Walker and Planner halt  

## Success Criteria
- no autonomous stride  
- no autonomous horizon projection  
- Mesh remains active but subdued  

Autonomy withdrawal prepares the organism for controlled dissipation.
